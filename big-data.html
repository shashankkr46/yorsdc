Big Data

OVERVIEW

  • As the world moves to a digital age, there is literally an explosion of data and big data frameworks makes it possible to stay on top of it. If a few years ago, megabytes and gigabytes used to be the extent of data, experts are now talking of several petabytes. Every major internet company - be it Google, Twitter, LinkedIn or Facebook - uses some form of big data frameworksto sort and search the vast amounts of data that their hundreds of millions of users are creating every second. Spark is free, open-source software written in Scala. Apart from Scala it also supports Java and Python. It makes a quick work of processing mountains of data by breaking it into smaller process that parallelly runs on multiple computers (cluster). Unlike Hadoop and Storm,Spark is a unified framework which deals with both batch data and real time data. It supports in memory processing and currently it’s the fastest big data framework in the market.
